## Problem

Traditional virtual machines can be heavy for student laptops. They take storage, need setup time, and can cause data loss when lab work is not mapped to the host system.

## My Contribution

I co-authored the abstract and research idea. I explained how Docker helps students reduce setup time, save storage, and keep lab files through volume mapping.

## Process / Methodology

1. Compared common VM setup pain points with a Docker-based setup.
2. Explained container images, containers, and mounted folders.
3. Documented how an Ubuntu container can support OS lab practice.
4. Covered safe container management and cleanup.
5. Presented Docker as a practical teaching tool for repeated lab work.

## Result / Output

The output was a research and innovation project showing Docker as a practical alternative for OS coursework environments.
